# XBot Project

## Overview

XBot is a comprehensive pipeline that ingests data from specified URLs, stores it in a LanceDB database with embeddings, and integrates with Twitter and OpenAI APIs to handle queries and respond via direct messages.

## Features

- **Data Ingestion:** Fetches and processes data from URLs.
- **Embeddings:** Generates embeddings using Sentence Transformers.
- **Database:** Stores and retrieves data using LanceDB.
- **OpenAI Integration:** Generates intelligent responses using GPT models.
- **Twitter Integration:** Sends responses via Twitter Direct Messages.
